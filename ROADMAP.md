# Roadmap

## Milestone 1

- repository structure
- architecture and specification docs
- stable interfaces for compiler, operators, runtime, and verifier
- graph execution for a minimal declarative IR

Goal: everything builds and a simple reasoning graph executes end to end.

## Milestone 2

- problem classification
- first adaptive compiler heuristics
- productionized initial operators
- richer runtime context and tracing

Goal: compile a problem into a graph and execute it reliably.

## Milestone 3

- benchmarks and fixed-pipeline baselines
- operator-level metrics and evaluation
- verification quality instrumentation

Goal: test whether adaptive compilation outperforms fixed workflows.

## Milestone 4

- optimization and learned compilation
- research artifacts and paper support
- open-source hardening and broader integrations

Goal: evolve C³ from prototype into a serious research platform.
